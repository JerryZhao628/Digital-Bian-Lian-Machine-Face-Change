## Overview
Digital Bian Lian explores the relationship between the human body and machine vision.  
Using a webcam and PoseNet, the system detects the user's face and overlays generative masks that change in response to gesture.

Inspired by the traditional concept of face-changing (Bian Lian), the work reinterprets it through algorithmic processes.  
The machine observes the body and continuously transforms the user’s identity in real time.

---

## Instructions (Operation Manual)
1. Open the project in p5.js editor or a browser  
2. Allow camera access  
3. Stand in front of the camera  
4. Raise your right hand above your shoulder  
5. Wave your hand to trigger a face change  

---

## Optional Blurb
The face is no longer fixed.  
It becomes a surface continuously rewritten by machine vision.

---

## Technical Details
- Built with p5.js and ml5.js (PoseNet)  
- Uses webcam input for real-time body detection  
- Detects facial keypoints (nose, eyes)  
- Uses gesture (hand above shoulder) to trigger interaction  
- Generates masks using simple shapes
